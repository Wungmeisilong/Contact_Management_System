# Contact_Management_System

一个基于控制台的简易通讯录管理系统，使用 C++ 实现，支持基础增删改查功能。

## 功能特性
- ✅ 添加联系人（姓名/性别/年龄/电话/地址）
- ✅ 显示全部联系人信息
- ✅ 删除指定联系人
- ✅ 按姓名查找联系人
- ✅ 修改联系人信息
- ✅ 清空通讯录

## 环境要求
- C++11 及以上编译器（GCC/Clang/MSVC）
- 标准 C++ 库
- 编译器VS2022
